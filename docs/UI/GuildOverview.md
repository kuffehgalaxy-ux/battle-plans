# UI-004 - Guild Overview

## Purpose

Provide officers with a live overview of guild participation during Territory Battles.

The Guild Overview is designed to replace manual checks inside the game by surfacing members who require attention.

---

## Key Metrics

- Members Complete
- Combat Only
- Deployment Only
- No Action
- Total GP Deployed
- GP Remaining
- Estimated Progress

---

## Member List

Members are grouped by activity status rather than alphabetically.

Priority order:

1. No Action
2. Combat Only
3. Deployment Only
4. Complete

---

## Quick Actions

Each member supports:

- View Details
- Send Reminder

---

## Success Criteria

An officer should identify every member requiring attention within 10 seconds.
