# UI-003 - Territory Battle Dashboard

## Purpose

Provide officers with a complete operational view of the current Territory Battle phase.

The dashboard replaces the need to repeatedly navigate between planets inside the game.

---

## Sections

### Header

- TB Name
- Current Phase
- Time Remaining

---

### Command Status

Displays:

- Current Status
- GP Remaining
- Members Remaining
- Recommended Action

---

### Planet Cards

Each active planet displays:

- Current Stars
- GP Progress
- GP Remaining
- Combat Mission Progress
- Deployment Progress
- Battle Plan Status

---

### Quick Actions

- Publish Orders
- Quick Nudge
- View Phase Report

---

## Design Goal

An officer should understand the current Territory Battle within 15 seconds.
