# Definitions of Done

This document defines what "complete" means for each major feature.

A feature is only complete when all acceptance criteria have been met.

---

## UI-003 - Territory Battle Dashboard

- Officer understands phase within 15 seconds.
- Current objective is obvious.
- Recommended action is displayed.
- Planet progress is easy to read.
- Mobile layout tested.
- Officer feedback incorporated.

---

## UI-001 - Command Centre

- Guild status visible immediately.
- Current event highlighted.
- Officer Inbox accessible in one tap.
- Recommended action displayed.
